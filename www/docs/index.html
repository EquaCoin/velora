🚀 Quick Start

Installation

# Clone the repository
git clone https://github.com/EquaCoin/velora.git
cd velora

# Build with Cargo
cargo build --release

# Run a file
./target/release/velora run example.vel

Hello World

main:
    print("Hello, Velora!")

🧬 Self-Hosting v11

Velora can generate code for itself using AI directives:

# AI: self aggiungi supporto per quantum computing

main:
    print(auto_evolved_feature())

This generates and saves code to generated.vel automatically.

📦 Package Manager

Install a package

vel add math-lib

Use in code

import "math-lib.vel"

main:
    print(square(5.0))
    print(factorial(10))

Create your own package

  1. Create a repository with lib.vel
  2. Add velora.toml manifest
  3. Create a GitHub release with tarball
  4. Submit to velora-packages registry

📝 Language Syntax

Variables

let x = 42           # Integer
let pi = 3.14159     # Float
let name = "Velora"  # String
let active = true    # Boolean

Functions

fn add(a: Int, b: Int) -> Int {
    return a + b
}

fn greet(name: String) -> String {
    return "Hello, " + name
}

Control Flow

if x > 10 {
    print("big")
} else {
    print("small")
}

while x > 0 {
    print(x)
    x = x - 1
}

Arrays and Maps

let nums = [1, 2, 3, 4, 5]
print(nums[0])        # Access
print(len(nums))      # Length

let user = {name: "Mario", age: 30}
print(map_get(user, "name"))

🧪 Testing

fn sum(a: Int, b: Int) -> Int {
    return a + b
}

test "sum works" {
    assert_eq(sum(2, 3), 5)
    assert_true(sum(0, 0) == 0)
}

main:
    print(sum(10, 20))

Run tests with: velora test file.vel
